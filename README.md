# 🏃 Mapa de Instalaciones Deportivas de Elche

Mapa interactivo que muestra instalaciones deportivas públicas, privadas y tiendas deportivas en Elche.

## 📋 Descripción

Este proyecto presenta un mapa interactivo desarrollado con Leaflet que permite visualizar y filtrar:

- **Instalaciones Municipales** (14): Polideportivos, piscinas, campos de fútbol, pabellones, etc.
- **Instalaciones Privadas** (442): Gimnasios, centros de artes marciales, piscinas privadas, clubs de pádel, etc.
- **Tiendas Deportivas** (81): Comercios especializados en artículos deportivos.

## ✨ Características

- 🗺️ Mapa interactivo con OpenStreetMap
- 🔍 Buscador por nombre o dirección
- 🎨 Filtros por tipo de instalación
- 🏷️ Filtros por categorías deportivas
- 📍 Marcadores diferenciados por color
- 💡 Información detallada en popups
- 📱 Diseño responsive

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript
- [Leaflet](https://leafletjs.com/) - Librería de mapas interactivos
- [OpenStreetMap](https://www.openstreetmap.org/) - Datos de mapas
